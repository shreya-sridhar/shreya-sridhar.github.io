Email ERA

Email ERA is an intelligent, self-learning end-to-end mail management tool that bridges the gap between mailbox and CRM using NLP capabilities. It reads through e-mails, processes content, categorizes them as per business use cases and generates auto-response suggestions to the majority emails. Developed a mobile responsive front-end application with ReactJS, Material-UI for CSS and MobX for state management. Also, the web application leverages the usage of WebSockets to fetch real-time email conversations.

Development year - 2019

Technology stack - React, Material-UI, MobX, WebSockets

Vision ERA OCR

Vision ERA platform uses OCR technologies to extract relevant data from printed documents such as government ID cards and invoices while automating a significant amount of cognitive work. Developed the front-end using HTML5 Canvas with ReactJS, Material-UI for CSS and MobX for state management. The uploaded document is mapped onto the canvas which shows the highlighted co-ordinates by hovering over the extracted data and also allowing users to extract a snippet by selecting the mouse over the area.

Development year - 2019

Technology stack - React, Material-UI, MobX
